To run the game simply double-click NinthBullet.py,
cd to the game's directory and type:

python NinthBullet.py

When running from terminal you can type -nofullscreen to disable fullscreen mode

CONTROLS:

	z to jump. Hold z down to jump higher
	x to shoot.
	Arrow keys to move
	Q or R to return to menu
	Escape to quit (at any time)
	Press Enter to select the highlighted choice while in menus
